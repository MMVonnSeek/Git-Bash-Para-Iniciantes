# 2. Comandos Básicos do Terminal (Bash)

O Git Bash não serve apenas para o Git; ele é um terminal completo que permite interagir com o seu sistema operacional usando comandos de texto. Dominar esses comandos é o primeiro passo para se sentir confortável na linha de comando.

## 1. Navegação Essencial

| Comando | Função | Exemplo de Uso |
| :--- | :--- | :--- |
| `pwd` | **P**rint **W**orking **D**irectory. Mostra em qual pasta você está. | `pwd` |
| `ls` | **L**i**s**t. Lista os arquivos e pastas no diretório atual. | `ls -l` (mostra detalhes) |
| `cd` | **C**hange **D**irectory. Muda para outra pasta. | `cd Documents` |
| `cd ..` | Sobe um nível na estrutura de pastas. | `cd ..` |
| `cd ~` | Volta para o seu diretório inicial (Home). | `cd ~` |

## 2. Gerenciamento de Arquivos e Pastas

| Comando | Função | Exemplo de Uso |
| :--- | :--- | :--- |
| `mkdir` | **M**a**k**e **Dir**ectory. Cria uma nova pasta. | `mkdir meu-projeto` |
| `touch` | Cria um novo arquivo vazio. | `touch index.html` |
| `cp` | **C**o**p**y. Copia arquivos ou pastas. | `cp arquivo.txt destino/` |
| `mv` | **M**o**v**e. Move ou renomeia arquivos/pastas. | `mv arquivo.txt novo-nome.txt` |
| `rm` | **R**e**m**ove. Exclui arquivos. | `rm arquivo-antigo.txt` |
| `rm -r` | Exclui pastas e seu conteúdo (**Cuidado!**). | `rm -r pasta-antiga` |

## 3. Visualização de Conteúdo

| Comando | Função | Exemplo de Uso |
| :--- | :--- | :--- |
| `cat` | Exibe o conteúdo de um arquivo diretamente no terminal. | `cat README.md` |
| `clear` | Limpa a tela do terminal. | `clear` |

## 4. Dica de Produtividade: O Tab

Use a tecla **Tab** para autocompletar nomes de arquivos e pastas. Isso economiza tempo e evita erros de digitação.

*   Exemplo: Se você digitar `cd Doc` e apertar **Tab**, o terminal completará para `cd Documents/`.
